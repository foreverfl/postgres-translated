# 머리말 {#preface}

이 책은 PostgreSQL의 공식 문서입니다. PostgreSQL 개발자들과 다른 자원봉사자들이 PostgreSQL 소프트웨어 개발과 병행하여 작성했습니다. 현재 버전의 PostgreSQL이 공식적으로 지원하는 모든 기능을 설명합니다.

PostgreSQL에 대한 방대한 정보를 관리하기 쉽게 만들기 위해, 이 책은 여러 부분으로 구성되어 있습니다. 각 부분은 서로 다른 사용자 계층 또는 PostgreSQL 경험의 다른 단계에 있는 사용자들을 대상으로 합니다:

- [Part 1](tutorial)은 새로운 사용자를 위한 비형식적인 소개입니다.

- [Part 2](sql)은 데이터 타입과 함수는 물론 사용자 수준의 성능 튜닝을 포함한 SQL 쿼리 언어 환경을 설명합니다. 모든 PostgreSQL 사용자는 이 부분을 읽어야 합니다.

- [Part 3](admin)는 서버의 설치와 관리에 대해 설명합니다. 개인용이든 다른 사람을 위해서든 PostgreSQL 서버를 운영하는 모든 사람은 이 부분을 읽어야 합니다.

- [Part 4](client-interfaces)는 PostgreSQL 클라이언트 프로그램을 위한 프로그래밍 인터페이스를 설명합니다.

- [Part 5](server-programming)은 서버의 확장 기능에 대한 고급 사용자를 위한 정보를 포함합니다. 사용자 정의 데이터 타입과 함수와 같은 주제들을 다룹니다.

- [Part 6](reference)는 SQL 명령, 클라이언트 및 서버 프로그램에 대한 참조 정보를 포함합니다. 이 부분은 명령어나 프로그램별로 정리된 구조화된 정보로 다른 부분들을 지원합니다.

- [Part 7](internals)는 PostgreSQL 개발자들에게 유용할 수 있는 다양한 정보를 포함합니다.
