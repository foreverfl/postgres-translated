# Part 4. SQL 언어 {#sql-language}

이 부분은 PostgreSQL에서의 SQL 언어 사용법을 설명합니다. SQL의 일반적인 구문을 설명하는 것으로 시작하여, 테이블을 생성하는 방법, 데이터베이스에 데이터를 채우는 방법, 그리고 데이터를 조회하는 방법을 다룹니다. 중간 부분에서는 SQL 명령에서 사용할 수 있는 데이터 타입과 함수들을 나열합니다. 마지막으로, 데이터베이스 튜닝에 중요한 여러 측면들을 다룹니다.

이 정보는 초보자가 처음부터 끝까지 따라가면서 너무 많이 앞으로 참조하지 않고도 주제들을 완전히 이해할 수 있도록 구성되어 있습니다. 각 장은 독립적으로 구성되어 있어서 고급 사용자들이 원하는 대로 개별 장을 읽을 수 있습니다. 정보는 주제별 단위로 서술형으로 제시됩니다. 특정 명령에 대한 완전한 설명을 찾는 독자들은 Part 6를 참조하시기 바랍니다.

독자들은 PostgreSQL 데이터베이스에 연결하고 SQL 명령을 실행하는 방법을 알고 있어야 합니다. 이러한 내용에 익숙하지 않은 독자들은 먼저 Part 1을 읽어보시기 바랍니다. SQL 명령은 일반적으로 PostgreSQL 대화형 터미널인 psql을 사용하여 입력하지만, 비슷한 기능을 가진 다른 프로그램들도 사용할 수 있습니다.
